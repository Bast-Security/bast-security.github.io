Bast

Access control has never been this easy.

Bast is an access control system with a wireless all-in-one smart-lock. Installation is a breeze and settings are changed with an easy-to-use app.

Setup In No Time

Completely set up a system in three easy steps.

  1. Install the lock on your door.
  2. Download the Bast App.
  3. Enter the passcode on the screen into the pin-pad of the lock.

Secure By Design

The server, locks, and app are all designed carefully and intentionally with security at the front of our minds.

  • Login and setup is passwordless, strengthening your account and simplifying your life.
  • The server is incapable of tracking user activity or controlling door states.
  • Communications use the latest encryption technologies.
  • Sane defaults and system architecture prevent misconfiguration.
  • Code is open-source to keep us in check.
